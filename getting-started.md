# Getting Started

This section covers the minimal prerequisites and a fast path to run Kong Hybrid Mode locally.

## Prerequisites
- Docker Engine (v20+)
- docker-compose v1.29+ or Docker Compose v2
- OpenSSL (for generating certificates)
- curl (for API tests)
- Optional: JDK 17 + Maven/Gradle to build sample services

See the proof-of-technology for the exact commands and a repeatable workflow.
